# cpp_rewriting_SPBGU
В file_h:
Классы
- <complex.h> - комплексные числа
- <fraction.h> - дроби.
- <polynomial.h> - полиномы
- <matrix.h> - матрицы, элементы линейной алгебры
- <Spline.h> - сплайны
Методы:
- <aproximate.h> - апроксимация
- <interpolators.h> - полиномиальная интерполяция[safe_integrate - рекомендуемая для использования функция]

- <defferencial_equation.h> - решения задачи Коши для уравнния разрешимого относительно старшей производной
- <integral.h> - численное интегрирование, поддерживает особенности на границе(внутри нет)
Отрисовка:
- <Array_xy_To_.h> файл с функциями отрисовки

Использование классов:
- <calc_computing_f.h> файл с матричным калькулятором
- <counting_methods_1.h> код численных методов(Метод простых итераций, Ньютона) и примеры запуска с примерами.\
nonlinear_system_with_simple_iterations, nonlinear_system_with_newton методы, остальное примеры применения

 реализации части кода в:
 _cpp_realisation_file

документация в
documents
